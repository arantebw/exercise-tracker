Exercise Tracker

Create A New User

POST /api/exercise/new-user

Add Exercise(s)

POST /api/exercise/add

Retrieve Exercises

GET /api/exercise/log?{userId}[&from][&to][&limit]

  • {} = Required
  • [] = Optional
  • from, to = Date (YYYY-MM-DD)
  • limit = Number
